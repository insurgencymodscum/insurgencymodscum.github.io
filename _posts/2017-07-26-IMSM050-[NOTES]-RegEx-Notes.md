Find: `CD [space] [number]`, e.g. "CD 1"

```
^CD\s+\d+\Z

string start
CD
whitespace
digit
string end
+ 1 or more
```
---

Total Commander Multi Rename Tool: Replace Blank Space

```
Search for: `\s`
Replace with: `-`
Check `RegEx`
```
