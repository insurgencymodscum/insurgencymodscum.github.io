LimeWire were sent a Cease-and-Desist letter by the RIAA, but instead of closing the client entirely as many others have done in the face of such notices, they have decided to filter copyrighted content within the client. If you use LimeWire to acquire copyrighted content, we recommend you do not upgrade. More details can be found on Slyck.com. A new P2P client based on LimeWire's source code has just been started, called FrostWire.
LimeWire Pro is basically the same client, but with a fancier look, no "nag screens", and it searches for new sources more often. If you want it, we suggest you compile it from source instead of paying for it.
Also, please note that the Gnutella network has nothing to do with the GNU Project. The Gnutella name was chosen randomly by it's author.

WinMX Peer Network

WinMX (http://www.winmx.com/) - Defunct/Deprecated - WINE - Proprietary
Frontcode Technologies, who wrote WinMX were sent a Cease-and-Desist letter from the RIAA, and closed the entire network by shutting down the peer cache servers that WinMX used for getting primary IPs. Some Italian coders created a workaround and set up their own peer cache servers - details can be found on Slyck.com.
To make WinMX work, download this file and place it in your WinMX folder. If there already is one, replace it.
If you don't know where to download WinMX, you can grab the last version ever released - 3.54 Beta 4 - (Standalone .exe file, no installer or other accompanying files) here.
