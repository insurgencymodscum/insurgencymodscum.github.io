# Word Wrap Shortcut (Notepad++ does not have this feature!) #

`Key Bindings - User`

```
[
    { "keys": ["ctrl+shift+w"], "command": "toggle_setting", "args": {"setting": "word_wrap"}}
]
```

# Multi-line editing #

```
CTRL+A // select all
CTRL+SHIFT+L // multi-line editing
HOME // move to beginning of line
```
