Starting this thread as a place too post open source dev programs and other useful tech stuff

-If u dont have photoshop Gimp is pretty sweet and even has some things photoshop doesnt i use em both
http://www.gimp.org/

-Gimpshop is a addon to gimp that makes it feel and operate more like photoshop havnt tried it sounds cool
http://www.gimpshop.com/

-blender open source 3d modeling program
http://www.blender.org/

-Nems Terrain gen useful for making very large 3d terrain fast
http://nemesis.thewavelength.net/index.php?c=59

-Gcfscape used to open gcf files in steam directory
http://nemesis.thewavelength.net/index.php?p=25

-Crafty ive never used this one b4 maybe sum1 can try it out seems to be a better model viewer then stock sdk one
http://nemesis.thewavelength.net/index.php?p=45

-vtfedit used to make/change/view vtf and vmt files for textures
http://nemesis.thewavelength.net/index.php?p=41

-VMEX lets u decompile .bsp files back into vmf fiels so u can open em in hammer
http://www.bagthorpe.org/bob/cofrdrbob/vmex.html

-Ardour never tried it but looks like an alright free open source sound editer
http://www.ardour.org/

-Audacity another sound program
http://audacity.sourceforge.net/

-File Zilla free ftp file sharing
http://filezilla-project.org/

-Firefox best open source browser if ur still using internet explorer or safari get firefox
http://www.mozilla.com/en-US/firefox/new/

-open office is an open source office program similar to microsoft office but free
http://www.openoffice.org/

-VLC player a free video player that plkays almost any video format
http://www.videolan.org/vlc/

-Inkscape vector graphics
http://inkscape.org/download/

-foobar simple to use and open source audio converter
http://www.foobar2000.org/

also need LAME encoder
http://lame.sourceforge.net/

-DOSbox a dos emulator for running old dos apps and games
http://www.dosbox.com/

-Zsnes snes emulator old school games on pc lol
http://www.zsnes.com/

-bittorrent need i say more
http://www.bittorrent.com/

-Avast free opensource anti virus, very nice and updates alot higly suggest this if u need anti virus for free
http://www.avast.com/index

-spybot removes spyware
http://www.safer-networking.org/index2.html

-TortoiseSVN Revision control software
http://tortoisesvn.tigris.org/

-CamStudio free screen capture software
http://camstudio.org/

-Process Explorer an in depth process utility give more info then hitting ctrl-alt-del. useful for finding out just whats running and where
http://technet.microsoft.com/en-us/sysinternals/bb896653
