Use "gpedit.msc" trick for local files.

Open the Control Panel
Open Internet Options
Click the Security Tab
Click on Local Intranet
Click on Sties
Click Advanced
Type the drive letter of your file server where the application is located in the “Add this website to this zone” box, e.g. "Z:\"
Click Add
Click Close
Click OK
Close Internet Options by clicking OK
Close the Control Panel
